# 常用Markdown語法

>字體

**粗體**　\
_斜體_ \
~~刪除線~~ 

```
**粗體**
_斜體_
~~刪除線~~
```

>標題

# 標題字
## 次標題字
### 小標題字
###### 小小標題字
```
# 標題字
## 次標題字
### 小標題字
###### 小小標題字
```

>列表

* Item 1
* Item 2
  * Item 2a
  * Item 2b
```
* Item 1
* Item 2
  * Item 2a
  * Item 2b
```

>階層式區塊

>123
>>123
>>>123
```
>123
>>123
>>>123
```

>小區塊 

`小區塊`
```
用``將文字前後包住
```

>大區塊
```
用```將文字上下包住
```
